# Mesto

Интерактивное приложение для обмена фотографиями с авторизацией, лайками и редактированием профиля. Финальная проектная работа на курсе Яндекс.Практикум.

---

## 📋 О проекте

Это финальный проект курса, в котором я впервые работал с REST API, модульной архитектурой и настроил сборку через Vite. Вся логика — от валидации форм до интеграции с сервером — написана самостоятельно.

**Особенности:**
- Регистрация и авторизация.
- Добавление, удаление и лайк карточек.
- Редактирование профиля и аватара.
- Валидация всех форм.
- Сборка Vite + автоматический деплой на GitHub Pages.
